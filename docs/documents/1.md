# Paste Support in Intunja CLI

## How to Paste Magnet Links

The CLI now supports **native clipboard paste** when adding magnet links or torrent paths.

### Paste Shortcuts by Terminal

| Terminal | Paste Shortcut |
|----------|----------------|
| GNOME Terminal | `Ctrl+Shift+V` |
| iTerm2 (macOS) | `Cmd+V` |
| Windows Terminal | `Ctrl+V` |
| tmux | Right-click or `Ctrl+B` then `]` |

### Steps to Add Magnet Link

1. **Copy** magnet link from browser
2. Press `m` in Intunja
3. **Paste** using your terminal's shortcut
4. Press `Enter`

The entire magnet link will paste correctly, including special characters.

### Input Features

- ✅ Full clipboard paste support
- ✅ Up to 500 character input
- ✅ Cursor navigation with arrow keys
- ✅ `Ctrl+U` to clear line
- ✅ `Ctrl+W` to delete word
- ✅ Automatic validation

### Troubleshooting

**Paste not working?**
- Verify your terminal's paste shortcut
- Try middle-click (Linux X11)
- Try right-click → Paste
- Manual typing also works

**Example magnet:**
```
magnet:?xt=urn:btih:1234567890ABCDEF&dn=Ubuntu&tr=http://tracker.example.com
```

The textinput component handles all standard terminal paste operations automatically.
